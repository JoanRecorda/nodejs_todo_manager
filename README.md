## Installation and Setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### To get the Lifecycle scripts
```
npm run
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Initialize the project
```
npm start
```
